With Soinnutator you can create nicely formatted chord sheets for songs. Source code.

Usage

Write the code for the song in the white box below this help, and press render to display the rendered document in the the white box above.

Syntax

The syntax is composed of groups of one or more lines delimited by an empty line.

When there are two lines, the top row is for chords and bottom row for lyrics. When the group has only one line, it is assumed to be the chord line. With two or more lines, the lines are aligned where there are two consecutive space and a non-space character.

A pipe character (|) has a special meaning. When pipe characters align on all lines, a vertical bar is displayed in their place. These are meant to represent bars in the song. It is not necessary to use them. 

                  | F                 | Em              | G        |
          This- _ | time I have real- | ly led my-self- | _ astray |

A broken pipe character (¬¶) works similarly, but displays as a dim dotted line. This is meant to be used for subdivision of the bar when necessary. For example, in a 4/4 song, you could divide the bar with 4 broken bars, when there is a chord change that occurs somewhere in the bar.

You can use a dollar sign ($) to add horizontal space. One $ is rendered as the width of a 0 (zero) character. This is mostly meant to be used to format chord sequencies when there are no lyrics.

At sign (@) is replaced with a non-braking space before rendering. (This might not be needed any more. You might want to use $ instead.)

# and b in chord names are automatically replaced with ‚ôØ and ‚ô≠, respectively, on rendering.

A line can also contain one of the following ‚Äôpragmas‚Äô.

#title Song name
Set the title of the song. Displayed as a first level heading.

#right Some text
Text displayed to the right of the first heading. For example, song writer or artist name.

#section Section name
Start a new section, for example verse or chorus. Lines of a section are kept together when possible in the printout. A section ends when another begins or at the end of text.

Hint: You can use your browsers developer tools to tweak the style. For example in Firefox press shift-F7 to open the Style Editor. Or press right mouse button over outputted element, select Inspect Element then Style Editor.

Rendering

The render button converts the inputted code to a rendered chord sheet.

Printing

Note that the rendered view does not accurately portray what the song looks like when printed. Use your browsers print preview to see how the end result really looks like.

Hint: To have the same size font on a A3 landscape sheet as on a A4 portrait sheet, scale the font to about 71 %.

Transponation

Transponation rotates sharp keys when going up and flat keys going down. So if you want to go to D‚ô≠ from C, you need to go up two units (from C to C‚ôØ to D) and then back one (from D to D‚ô≠).
